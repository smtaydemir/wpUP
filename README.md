![cat](https://github.com/smtaydemir/wpup/blob/master/intro.gif?raw=true)

# Install 

## Composer

```
composer global require "smtaydemir/wpup=dev-master"
```

```
export PATH=~/.composer/vendor/bin:$PATH
```

```
$ wpup new <blogname>
```

## Github

```
git clone https://github.com/smtaydemir/wpup.git
```

```
$ cd <folder>
```

```
$ ./wpup new <blogname>
```
